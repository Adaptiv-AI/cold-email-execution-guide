Success KPIs

We do NOT track open rates for clients. It hurts deliverability. Focus on these metrics:

Stage Metric Name Target Goal Action if Failed
1. Tech Mailbox Health 100% Pause client campaign. Warm up for 14 days.
1. Tech Blacklist Status Clean Isolate the infected domain.
2. Data Bounce Rate < 2% Re-verify the client's list. Do not use "Risky".
3. Copy Reply Rate 1% - 5% If low (<1%), the client's offer is weak. Rewrite Body.
3. Copy Positive Reply % > 50% of replies If leads are angry, check if targeting matches the offer.
4. Scale A-Variable < 200 Optimize the offer to book calls with fewer emails.
Critical Compliance

ðŸš« Safety Protocols

Violating any of these rules puts the client's reputation at risk.

1. NO LINKS IN EMAIL 1

Never include a URL (client website, calendar, YouTube) in the first email. It triggers spam filters instantly. Only send links after the lead replies.

2. NO OPEN TRACKING

Ensure the "Track Open Rate" setting is OFF in the client's Instantly campaign. This uses an invisible pixel that hurts deliverability.

3. NEVER USE CLIENT'S MAIN DOMAIN

CRITICAL: Never send cold emails from the client's primary domain (e.g., @clientcompany.com). Only purchase and use similar "burner" domains (e.g., @try-clientcompany.com).

4. NO UNSUBSCRIBE LINKS

Do not use the automatic "Unsubscribe" footer link. It looks like marketing spam. Instead, use natural language: "PS: Let me know if I'm barking up the wrong tree."

5. NO AGGRESSIVE SCALING

Do not send more than 30-50 emails per mailbox per day for any client. Scaling requires buying them more mailboxes, not increasing volume per mailbox.

Operations Phase 01

Tech Setup

Goal: Build a fortress around the client's main brand.

1. Buy "Burner" Domains
Buy 2-3 secondary domains that look like the client's brand (e.g., get-clientname.com). Forward them to the client's actual website.
2. Create Google Workspaces
Create 2-3 inboxes per domain. Use real names from the client's team or a generic persona (e.g., "Sarah at ClientCo").
3. Add DNS Records (The Security Pass)
  • SPF: v=spf1 include:_spf.google.com ~all
  • DKIM: Generate via Google Admin.
  • DMARC: v=DMARC1; p=quarantine;
4. Connect to Instantly.ai
Enable Warm-up. Settings: Increase by 2 per day, Limit to 25. Wait 14 days before launching client campaign.
Operations Phase 02

Lead Sourcing

Goal: Find leads that match the Client's ICP (Ideal Customer Profile).

1. Filter Search (Apollo)
Input Client's criteria: Job Titles, Location, Company Size, Keywords.
2. Clean Data (Million Verifier)
Upload CSV to verifier. Only download "Valid/Good" emails. Do not risk the client's domains on "Risky" emails.

*Note: Sending to bad data causes bounces, which gets domains blacklisted. Always clean the client's list.

Operations Phase 03

Copywriting Strategy

Goal: Start a conversation for the client. Do not pitch. Sell the "next step."

The "Triple Tap" Structure

1. The Preview (Subject + Line 1)
Subject: "Question for {{firstName}}" (Boring wins).
Line 1: "Hey {{firstName}}, just wondering if you are handling [Client's Service Area]?"
2. The Body (Value)
State the client's value prop in 6th grade reading level.
"We help [Target Industry] solve [Specific Problem] without [Pain Point]."
3. The CTA (Low Friction)
NO LINKS ALLOWED. Ask for interest.
"Mind if I send over a quick video explaining how?"

Spintax (Anti-Spam)

Apply Spintax in Instantly
Ensure variations are set so the client's message is unique for every send.
{{Hi | Hello | Hey}} {{firstName}},
Operations Phase 04

Launch & Management

Sending Rules Checklist

1. Tracking Settings OFF
Ensure "Open Tracking" is disabled in the client's Instantly settings.
2. Text Only Mode
Ensure "Send as Plain Text" is enabled. No HTML, no fancy signatures with logos.

Daily Operations

1. Monitor Client Unibox
Reply to positive leads within 10 minutes. THIS is when you send the client's calendar link.
2. Remove Bounces
If the client's campaign exceeds 2% bounce rate, PAUSE immediately and re-verify data.
